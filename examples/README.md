# Anabelle Demo

See [DEMO](https://examples.contributte.org/packages/anabelle/).

## Installation

```bash
composer install
```

## Generating API Documentation

```bash
composer generateDocs
```

or

```bash
vendor/bin/anabelle docs-src docs --overwriteOutputDir
```

## View documentation

```bash
php -S localhost:8000 -t docs
```
